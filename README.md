# DATA ENGINEERING

This repository is for a quick installation and configuration of software using Docker.

## Docker commands

Listing all images
```
docker images ps
```

Docker run "centos". Running a "centos" image.  

```
docker run centos bash
```

Starting a "centos" container.

```
Docker start "centos".
```
